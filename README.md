A modified version of [**Felineification 16x**](https://modrinth.com/resourcepack/felineification-16x).

## ✨ Features
- Connected hotbar
- Edited menu buttons
- Less distinguishable ore outlines
- Mod support
- No "loading" breaking animation
- No short swords
- Vanilla-like apples
